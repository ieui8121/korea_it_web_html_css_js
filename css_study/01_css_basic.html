🎨 인라인 스타일과 내부 스타일 비교하기

이 단락은 인라인 스타일이 적용되었습니다. 내부 스타일 시트에서 'p' 태그를 초록색으로 지정했지만, 태그에 직접 파란색(blue)을 지정하니 그 규칙을 덮어쓰고 파란색으로 표시됩니다.

정리: 스타일 적용 우선순위

하나의 태그에 내부 스타일과 인라인 스타일이 모두 적용될 수 있을 때, 브라우저는 아래와 같은 우선순위를 따릅니다.

  • 1순위: 인라인 스타일 (태그에 직접 적용)
  • 2순위: 내부 스타일 시트 (<style> 태그)

결론적으로, 더 구체적이고 가까이에서 지정한 '인라인 스타일'이 '내부 스타일'보다 우선 적용됩니다.
